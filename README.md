# Study Algorithm in Java

This repository contains my study notes and implementations of common algorithms in Java.  
The code is organized by algorithm type under the `src` folder.

## Implemented Algorithms

- **BubbleSortStudy** – Bubble Sort algorithm implementation
- **InsertionSortStudy** – Insertion Sort algorithm implementation
- **SelectionSortStudy** – Selection Sort algorithm implementation
- **QuickSortStudy** – Quick Sort algorithm implementation
- **MargoSortStudy** – Merge Sort algorithm implementation (typo in folder name, actually Merge Sort)
- **DijkstraStudy** – Dijkstra's shortest path algorithm implementation


## Project Structure
```
src/
├─ BubbleSortStudy/
├─ InsertionSortStudy/
├─ SelectionSortStudy/
├─ QuickSortStudy/
├─ MargoSortStudy/
├─ DijkstraStudy/
```
